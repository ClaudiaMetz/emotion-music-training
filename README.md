# Emotion Music – Training

Este repositorio contiene la fase de entrenamiento y experimentación del proyecto
**“Ponele música a tus emociones”**.

El proyecto fue desarrollado originalmente como trabajo académico en Google Colab, con el objetivo de entrenar un modelo de Deep LLearning para la clasificación de emociones a partir de imágenes faciales.

## Alcance de este repositorio

- Entrenamiento del modelo
- Experimentación y pruebas
- Desarrollo en entorno notebook (research)

Este repositorio **no está orientado a producción**.

## Estado actual

- Entorno experimental
- El modelo fue entrenado y evaluado en Colab
- En esta etapa no se persistieron los pesos como artefacto versionado

## Próxima etapa

La migración del modelo a una arquitectura de serving productiva (Docker + Vertex AI) se documenta en el repositorio `emotion-music-mlops`.
